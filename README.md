# 🧠 Cogito – AI-Powered Task Manager  

Cogito is a **minimalist AI task manager** that helps users stay organized by **automatically generating, prioritizing, and scheduling tasks** based on their needs. Designed with **SwiftUI** and powered by **OpenAI**, Cogito ensures a smooth and intelligent productivity experience.  

---

## 🚀 Features  

✅ **AI Task Suggestions** – Generate tasks based on importance and urgency 🤖  
✅ **Smart Scheduling** – AI organizes tasks efficiently to optimize daily workflow 📅  
✅ **Minimalist UI** – Distraction-free, elegant interface for better focus 🎯  
✅ **Dark & Light Mode Support** – Adaptive colors for seamless usability 🌗  
✅ **Distraction Alerts** – Detects long inactivity periods and suggests focus sessions ⚡  
✅ **Sync Across Devices** – Works across **iPhone, iPad, and Mac** for a unified experience 🔄  
✅ **SwiftUI Widgets** – Quick access to tasks directly from the home screen 🏡  
✅ **Secure Data Storage** – Local and cloud-based storage with **CoreData / CloudKit** 🔒

## 🛠️ Tech Stack  

- **SwiftUI** – Modern, declarative UI framework  
- **OpenAI API** – AI-powered task generation  
- **GraphQL Backend** – Efficient data fetching and updates  
- **CoreData / CloudKit** – Secure, offline-friendly data storage  
- **Combine & Async/Await** – Reactive programming for smooth performance
